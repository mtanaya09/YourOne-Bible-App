
# YourOne Bible App
## Getting Started
### Pre-requisites
- [ ] Node.js 14.18.1
- [ ] Expo
- [ ] Expo CLI
- [ ] Yarn
1. `git clone` the repository.
2. `yarn` to get all the required modules.
3. `expo start` or `yarn start` to run.
